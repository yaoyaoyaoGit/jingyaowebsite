公司简介

上海晶耀光电科技有限公司是中国汽车工业进出口有限公司投资控股的子公司成立于 2007年。是专业生产车用LED车灯模块、LED车用产品的高新技术企业。公司凭借 IATF16949:2016质量管理体系，先后开发生产了LED汽车日间行车灯、LED远近光一体大灯 模组、LED单远光、LED单近光组合透镜模组、转向灯、刹车灯、牌照灯、尾灯组合灯、仪 表灯、内饰氛围灯等LED车灯系列产品。为国内汽车、摩托车主机厂提供优质配套服务。
